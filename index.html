Core projects

The two projects that anchor my resume — one systems research, one backend engineering.

Embedded Systems

Smart RFID Access Control with Face Capture

Distributed authentication logic across three microcontrollers — an Arduino Uno handles RFID card classification and relays results over UART to an 8051, which drives the servo lock, I2C LCD, and LED/buzzer outputs. Solved stale-frame capture by triggering a full ESP32-CAM deinit/reinit cycle on each intrusion event, exposing the latest intruder photo over a self-hosted Wi-Fi endpoint with IPFS-ready architecture for future audit logging.

Arduino Uno 8051 ESP32-CAM MFRC522
Automation

AutoNova Parts — Inventory Stock Alert System

Eliminated manual restocking with a UiPath bot that polls the admin dashboard every 15 seconds, fires supplier emails via Gmail SMTP, sends WhatsApp confirmations, and clicks the ML restock button autonomously — zero human intervention after checkout. Engineered a browser-only ensemble forecasting engine (EMA + linear trend + velocity scoring) to predict 7-day part demand and auto-generate PDF purchase orders, syncing admin and storefront views in real time via BroadcastChannel with no backend server.
